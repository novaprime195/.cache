# Singly Linked List in C (Dockerized)

A simple C program demonstrating singly linked list operations, containerized with Docker.

## Features
- Insert at beginning
- Insert at end
- Delete node by value
- Display list
- Memory management

## Quick Start

### Build Docker Image
```bash
docker build -t linked-list .
```

### Run Container
```bash
docker run --rm linked-list
```

## Local Compilation (without Docker)
```bash
gcc -o linked_list linked_list.c
./linked_list
```

## Output
The program demonstrates basic linked list operations with sample data.
